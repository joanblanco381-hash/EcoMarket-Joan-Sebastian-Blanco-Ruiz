# EcoMarket-Joan-Sebastian-Blanco-Ruiz

## EcoMarket 🌱
## Descripción

EcoMarket es una página web desarrollada con HTML y CSS que tiene como objetivo promover la venta de productos naturales y ecológicos. La interfaz ofrece una navegación sencilla y un diseño limpio enfocado en la sostenibilidad y el cuidado del medio ambiente.

## Características
 -Página de inicio con sección principal (Hero).
 -Barra de navegación.
 -Logo de la empresa.
 -Botón de compra rápida.
 -Diseño responsivo y moderno.
 -Temática ecológica con colores naturales.
 -Tecnologías utilizadas:
    >HTML5
    >CSS3
